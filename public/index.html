What is this?

Currently the only way to find deadlines for Parliamentary Oral Questions is this PDF file. Many MP staff members spend time copying the entries over to their MP's calendars manually.
This tool provides a subscribable calendar which updates daily with the latest published deadlines.
Anyone is welcome to use it - the code is all public so you can even check how it works.



How do I use it?

With your calendar tool (probably Outlook), you need to subscribe to this calendar url: 

https://tabling.simmsreeve.com/ical
To get setup, please follow the "Workaround" instructions here.
